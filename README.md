# Тестовое задание
Чтобы запустить проект, нужно запустить файл main.py
Создается сервер на порту 8080
Сервер:
http://127.0.0.1:8080

Для работы программы нужны следующие библиотеки:
* selenium
* BeautifulSoup
* sqlalchemy
* uvicorn
